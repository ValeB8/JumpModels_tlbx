**************************************************************************************************************************************************************************
The toolbox can only be fully used with CVX for Matlab installed. Please install the CVX package!
**************************************************************************************************************************************************************************

ADD to the path:
    - cvx folder (running cvx_setup.m and cvx_startup.m)
    -JumpModel_tlbx

* 'Data Electric Power consumption' containes the data for the test on the reconstruction of electric consumption patterns (Sections 4.1.2 and 4.2.2)
   and a function to build the datasets used for training.

* 'Learning Rarely Jump Models' containes the general code to learn RJM and to reproduce the Examples in Section 4.1 

* 'Learning Markov Jump Models' containes the general code to learn MJM and to reproduce the Examples in Section 4.2
   
*************************************************************************************************************************************************************************
Compiled code only for MAC. Run JumpModel_compile.m to compile code!
